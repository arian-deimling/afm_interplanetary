 I. Sign Up
Figure 5.1.1: Signup Form
# Action Event
1 printable character input Validate username input field and display message to user if field value is invalid.
2 Validate password input field and display message to user if field value is invalid.
3 Validate password verification input field and display message to user if field value is invalid.
6 Validate security quesiton answer input field and display message to user if field value is invalid.
4 left-mouse click Toggle checkbox. If checkbox is checked, set password field to plain-text; otherwise, set password field to hidden-text.
5 select question Validate security quesiton selection field and display message to user if field value is invalid.
1, 2, 3, 6 <ENTER> keypress Validate username, password, password verification, security question, & security question answer input fields and display message to user if any field value is invalid; otherwise, POST field values to the server.
7 left-mouse click
N/A response from server If response from server is OK, redirect user to Login Page; otherwise, display server error message to user.
